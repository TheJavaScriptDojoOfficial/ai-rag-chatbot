# Web (Next.js Frontend)

Next.js frontend for the Company RAG Chatbot. App Router, TypeScript, Tailwind CSS, ESLint. RAG and chat UI will be added in later phases.

## Structure

- `src/app/layout.tsx` — root layout
- `src/app/page.tsx` — home page
- `src/app/globals.css` — global styles

## Run (Mac / Linux)

```bash
cd apps/web
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Requirements

- Node.js 18+
